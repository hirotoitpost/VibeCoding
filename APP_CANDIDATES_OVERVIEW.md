## 📊 難易度レベル定義

### ⭐ 初級 (Beginner)
- **特徴**: 単一言語、シンプルな API、ローカル実行
- **推定期間**: 0.5-1 セッション
- **学習ポイント**: 言語基礎、CLI、ファイル I/O
- **例**: 天気情報ツール (ID 005)

### ⭐⭐ 中級 (Intermediate)  
- **特徴**: フルスタック (Frontend+Backend), 複数ライブラリ, Docker
- **推定期間**: 1-2 セッション
- **学習ポイント**: API 設計, テスト, コンテナ化, E2E 検証
- **例**: Web家計簿 (ID 008-009), MQTT (ID 011)

### ⭐⭐⭐ 上級 (Advanced)
- **特徴**: マイクロサービス, 複数ドメイン, リアルタイム機能, 外部 API 統合
- **推定期間**: 2-3 セッション
- **学習ポイント**: 分散システム, メッセージバス, ブロックチェーン, トランザクション
- **例**: スマートホーム IoT (ID 013), DApp (ID 014-015)
