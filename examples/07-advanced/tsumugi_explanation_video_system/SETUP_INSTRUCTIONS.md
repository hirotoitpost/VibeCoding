# 立ち絵＆音声合成リアルタイム連動システム セットアップ指示書（春日部つむぎ統一版）

## 1. プロジェクト概要
Windowsデスクトップ上に「春日部つむぎ」の立ち絵を背景透過で配置し、マイク入力またはテキスト入力に合わせて「VOICEVOX：春日部つむぎ（Speaker ID: 14）」の音声で喋らせ、口パク（リップシンク）をリアルタイムで連動させる環境を構築する。

## 2. 必要ツール一覧
AIエージェントは以下のツールのインストール状況を確認し、未インストールの場合は入手先を提示すること。

| ツール名 | 役割 | 備考 |
| :--- | :--- | :--- |
| **VOICEVOX** | 音声合成エンジン | 「春日部つむぎ」をAPIサーバー(Port:50021)として使用。 |
| **ゆかりねっとコネクターNEO** | 司令塔（音声認識・連動） | マイク入力をVOICEVOXへ橋渡しし、字幕出力等も行う。 |
| **Shoost** | 立ち絵表示・操作 | 春日部つむぎのPSDを透過表示。回転・反転・口パク制御。 |
| **春日部つむぎ公式立ち絵** | ビジュアル素材 | 公式配布のPSDファイル（PSDTool対応版を推奨）。 |

## 3. ユーザー作業が必要な項目（手動操作）
1. **VOICEVOXの起動:** バックグラウンドで起動を確認。
2. **立ち絵素材の配置:** 春日部つむぎのPSDファイルを専用フォルダに格納。
3. **Shoostへの読み込み:** PSDをドラッグ＆ドロップし、キャラのサイズや位置をデスクトップ上に固定。

## 4. 環境変数の設定
AIは以下の変数を使用してスクリプトや設定ファイルを生成すること。

* `VOICEVOX_ENGINE_URL`: `http://localhost:50021`
* `TSUMUGI_ASSET_DIR`: 春日部つむぎの立ち絵資産ディレクトリ
* `VOICE_CHARACTER_ID`: `14` (春日部つむぎ ノーマル)

## 5. セットアップ用 PowerShell スクリプト（自動生成用）
AIエージェントは、以下の機能を持つ `setup_env.ps1` を生成すること。

* **ディレクトリ入力:** `Read-Host` でツールのインストール先や素材パスをユーザーに問い合わる。
* **パスの自動作成:** 入力されたパスにフォルダを作成し、エクスプローラーで開く。
* **環境変数の登録:** `[Environment]::SetEnvironmentVariable` で上記4の変数を「ユーザー」レベルで永続化。
* **起動ランチャーの作成:** 全ツールをワンクリックで起動する `.bat` または `.ps1` をデスクトップに出力。

## AIエージェントへのタスク依頼

**Step 1:** 上記仕様に基づき、ユーザーにパスを入力させて環境変数をセットする PowerShell スクリプトを作成してください。
**Step 2:** 春日部つむぎ（ID:14）に対して「準備完了だよ！」と喋らせるテスト用の PowerShell (curl) スクリプトを作成してください。
**Step 3:** Shoostで「音声入力デバイス」として自身のスピーカー（または仮想ミキサー）を選択し、VOICEVOXの音に反応して口が動くように設定する手順をまとめてください。
