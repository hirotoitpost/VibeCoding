# Shoost 設定ガイド - 春日部つむぎリップシンク連動セットアップ

## 概要
Shoost は VOICEVOX の音声データに反応してキャラクターの口を動かす（リップシンク）ツールです。  
このガイドでは、Shoost で春日部つむぎの立ち絵を読み込み、音声入力デバイスから VOICEVOX の音に反応するように設定するステップを説明します。

---

## 前提条件

### ✅ 以下が完了していることを確認してください：
1. **VOICEVOX** が起動しており、APIサーバー（`http://localhost:50021`）として動作
   - ターミナルで確認: `curl http://localhost:50021/health`
   
2. **春日部つむぎの立ち絵素材**（PSD ファイル）
   - 推奨: 公式サイトから配布の PSD ファイル
   - Shoost 対応の PSD がベスト（レイヤー構成が最適化済み）

3. **Shoost** がインストール済み
   - インストール先は `setup_env.ps1` で設定したパス

---

## セットアップ手順

### **Step 1: Shoost を起動**

1. デスクトップの `launch_tsumugi_system.bat` をダブルクリック
   - または手動で Shoost.exe を実行

2. Shoost ウィンドウが開く
   - 初期状態は真っ黒（キャラクター未読み込み）

---

### **Step 2: 春日部つむぎの立ち絵を読み込む**

#### 方法 A: ドラッグ&ドロップ（推奨）

1. **エクスプローラー**を開く
   - `setup_env.ps1` で設定した `TSUMUGI_ASSET_DIR` フォルダを開く
   - 例: `C:\Users\YourName\Tsumugi\Assets`

2. **PSD ファイルをドラッグ**
   - 春日部つむぎの PSD ファイルを見つける
   - Shoost のウィンドウにドラッグ&ドロップ

3. **立ち絵が表示される**
   - 数秒待機
   - 春日部つむぎがデスクトップに表示される ✅

#### 方法 B: メニューから読み込み

1. Shoost のメニュー → `ファイル` → `立ち絵を開く`
2. PSD ファイルを選択
3. `開く` をクリック

---

### **Step 3: 立ち絵のサイズ・位置を調整**

1. **マウスでドラッグ**
   - 立ち絵をドラッグして位置を調整

2. **リサイズハンドル**
   - 端または角をドラッグしてサイズ変更
   - Ctrl + スクロール でもリサイズ可能

3. **背景の透過確認**
   - Shoost は既に背景透過です ✅
   - デスクトップ背景が見えているはず

---

### **Step 4: 音声入力デバイスの設定**

#### 前提: 仮想オーディオデバイスの準備

Shoost が VOICEVOX の音に反応するには、**Shoost が聞き取る音源**を設定する必要があります。

**推奨構成:**

```
VOICEVOX (出力)
    ↓
仮想ミキサー（例: VB-Audio Virtual Cable）
    ↓
Shoost（入力デバイスとして設定）
```

---

#### 🔧 仮想ミキサーのセットアップ

##### オプション 1: **VB-Audio Virtual Cable**（無料、推奨）

1. **ダウンロード**: https://vb-audio.com/Cable/
   - Windows 向けの無料版をダウンロード

2. **インストール**:
   - インストーラーを実行
   - 再起動（強制要求あり）

3. **VOICEVOX の出力先を変更**:
   - VOICEVOX 設定 → `音声出力`
   - `CABLE Input (VB-Audio Virtual Cable)` を選択

4. **Windows スピーカーでも聞きたい場合**:
   - VB-Cable 出力を Stereo Mix に接続（Windows 設定）
   - または別のミキサーアプリを使用

---

##### オプション 2: **Stereo Mix**（Windows 内蔵、環境依存）

⚠️ Stereo Mix は Windows のオーディオデバイスによって有効・無効が異なります。

1. **Windows タスクバー** → サウンド設定
   - 音量ミキサー → プライバシー設定
   - マイクの許可 ✓

2. **Stereo Mix を有効化** (デバイスによっては利用不可)
   - オーディオデバイス設定から確認

---

#### Shoost での音声入力デバイス設定

1. **Shoost のメニュー** → `設定` または `Options`
   - オーディオセクションを探す

2. **入力デバイス（Microphone/Audio Input）**:
   - `CABLE Output (VB-Audio Virtual Cable)` を選択
   - または `Stereo Mix` を選択

3. **感度（Sensitivity）調整**:
   - スライダーを `中程度` に設定
   - テスト: VOICEVOX で音声を合成 → 立ち絵の口が動く

4. **音声レベルのテスト**:
   - `Test` または `Check Levels` ボタンで確認
   - レベルメーターが妥当な範囲か確認

---

### **Step 5: 口パク（リップシンク）の詳細設定**

#### 口の形状を制御するレイヤー

Shoost 対応の PSD では、通常以下のレイヤー構成になっています：

```
春日部つむぎ.psd
├── 口 (Group)
│   ├── ニュートラル (neutral)
│   ├── あ行 (a)
│   ├── か行 (ka)
│   ├── さ行 (sa)
│   ├── た行 (ta)
│   ├── な行 (na)
│   ├── は行 (ha)
│   ├── ま行 (ma)
│   └── や行 (ya)
├── 目 (Group)
├── 眉毛 (Group)
└── 体 (Group)
```

#### Shoost での口パク設定

1. **Shoost メニュー** → `レイヤー設定` または `Layer Mapping`

2. **音声認識フォネームマッピング**:
   - Shoost が音声を周波数分析して、各フォネムを判定
   - 自動マッピング: `Auto Map` または `Auto Detect` をクリック

3. **手動調整**（オプション）:
   - 各フォネムに対応するレイヤーを割り当て
   - 例：`/a/` フォネム → `あ行` レイヤー表示

4. **Test Mode** でテスト:
   - VOICEVOX で「あああああ」と発音させる
   - → `あ行` レイヤーが表示される
   - → OK ✅

---

### **Step 6: 目と眉毛の動作設定（オプション）**

#### 瞬きの設定
1. **Shoost メニュー** → `表情` または `Expression`
2. **瞬きのレイヤー** を設定
3. **瞬き周期** を調整（例：3〜5秒ごと）

#### 眉毛の上下
1. **感情パターン** を設定（オプション）
2. 通常時・疑問時・喜び時など

## 🐛 トラブルシューティング

### **Q1: 立ち絵が暗い・見えない**

**症状**: Shoost ウィンドウが黒いまま

**解決策**:
1. PSD ファイルが正しく読み込まれているか確認
2. Shoost のメニュー → `明度調整` で明るさを上げる
3. PSD ファイル自体の互換性確認（Shoost 対応版か）

---

### **Q2: 口が全く動かない**

**症状**: VOICEVOX が音声を出力しても、立ち絵の口が動かない

**原因と対策**:
1. **音声入力デバイスが設定されていない**
   - Shoost の設定 → 入力デバイス確認
   - CABLE Output / Stereo Mix が選択されているか

2. **VOICEVOX の出力先が間違っている**
   - VOICEVOX の設定 → 音声出力
   - スピーカー が選択されていないか → CABLE Input に変更

3. **ボリュームが小さい**
   - Windows ボリュームミキサーで確認
   - VOICEVOX のボリュームが 0 になっていないか

4. **Shoost の感度が低すぎる**
   - Shoost の設定 → Sensitivity スライダーを上げる

---

### **Q3: 音が出ない・スピーカーから聞こえない**

**症状**: VOICEVOX は起動しているが音が出ない

**原因と対策**:
1. **出力デバイスの確認**
   - Windows タスクバー → スピーカーアイコン
   - 正しいスピーカーが選択されているか

2. **CABLE の場合、Windows スピーカーに接続されているか**
   - VB-Audio Software → Voicemeeter 等で確認
   - CABLE 出力を Stereo Mix / Speakers に接続

3. **マイクの入力が無効になっていないか**
   - Windows 設定 → プライバシー → マイク ✓ 有効

---

### **Q4: ゆかりねっと NEO と連携しない**

**症状**: マイク入力がゆかりねっト NEO から VOICEVOX に送られない

**原因と対策**:
1. **ゆかりねっと NEO の設定確認**
   - プラグイン → VOICEVOX プラグイン有効化
   - VOICEVOX URL: `http://localhost:50021` 設定確認

2. **マイクデバイスの確認**
   - ゆかりねっと NEO: `設定` → `入力デバイス` 確認
   - 正しいマイクが選択されているか

3. **出力先の確認**
   - ゆかりねっと NEO: `出力先` = `VOICEVOX` 設定
