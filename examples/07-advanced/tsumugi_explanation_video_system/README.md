# Tsumugi Explanation Video System (Spring日部つむぎ 解説動画システム)

## ⚠️ このプロジェクトは DEPRECATED (廃止) です

---

## 廃止理由

### 問題点
- **PSD ファイル非対応**: Shoost は PDF 形式に対応していません
- **当初想定と異なる**: PSD 立ち絵を Shoost で読み込める前提は誤りでした

### 代替案
新しいプロジェクト「AviUtl + PSDToolKit + VOICEVOX 統合」に移行予定

詳細は [IMPLEMENTATION_STATUS.md](./IMPLEMENTATION_STATUS.md) を参照してください。
