### 必要なツール
- ✅ VOICEVOX v0.25.1 以上
  - ダウンロード: https://voicevox.hiroshiba.jp/
  - OS: Windows / macOS / Linux 対応

- ✅ テキストエディタ（VS Code など）

- ✅ PowerPoint 2016以上

### 方法 3: VOICEVOX API 使用（上級向け）

VOICEVOX は Web API を公開しているため、
Python/JavaScript で自動化可能。

```python
import requests
import json

# VOICEVOX API エンドポイント
API_URL = "http://localhost:50021"

def generate_audio(text, speaker_id, output_path):
    # テキストから音声パラメータ生成
    params = {
        "text": text,
        "speaker": speaker_id
    }
    
    response = requests.post(
        f"{API_URL}/audio_query",
        params=params
    )
    
    # 音声合成
    audio_response = requests.post(
        f"{API_URL}/synthesis",
        json=response.json(),
        params={"speaker": speaker_id}
    )
    
    # ファイル保存
    with open(output_path, 'wb') as f:
        f.write(audio_response.content)

# 使用例
generate_audio(
    "こんにちは。今日は...",
    speaker_id=0,  # ナレーター
    output_path="Part_A/S01_intro_title.wav"
)
```

**メリット**: 完全自動化 / 24 個のセリフを数分で生成

**所要時間**: 5～10 分（セットアップ含む）

## 🔧 トラブルシューティング

### 症状 1: VOICEVOX で日本語入力ができない

**原因**: テキストエンコーディング

**解決**:
1. テキストファイル（.txt）を UTF-8 で保存
2. VOICEVOX にコピペ前に確認

```powershell
# PowerShell で確認
Get-Content "file.txt" -Encoding UTF8
```

---

### 症状 2: 音声ファイルが PowerPoint に埋め込めない

**原因**: ファイル形式 / ビット深度の不一致

**解決**:

VOICEVOX の出力設定を確認：
- ✅ フォーマット: WAV
- ✅ ビット深度: 16bit
- ✅ サンプリングレート: 44.1kHz

```bash
# ffmpeg で変換
ffmpeg -i input.wav -acodec pcm_s16le -ar 44100 output.wav
```

---

### 症状 3: 再生時に音が出ない

**原因**: サウンド設定 / OS 設定

**解決**:
1. PC のボリュームを確認
2. PowerPoint の「スライドショーの設定」をリセット
3. 別のスピーカー / ヘッドホンで確認

---

### 症状 4: 複数話者の音が混ざっている

**原因**: ファイル統合時の失敗

**解決**:
- 異なる話者は**別ファイル**で生成
- PowerPoint では時系列に埋め込み（自動再生 ON）
