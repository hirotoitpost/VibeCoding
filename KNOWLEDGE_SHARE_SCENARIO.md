## 【スライド 3】Example 01 - コードハイライト

```
┌──────────────────────────────────────┐
│  天気ツール - キー実装              │
│                                      │
│  📌 キャッシング実装                │
│  ┌────────────────────────────────┐ │
│  │ cache = {}                      │ │
│  │ CACHE_TTL = 300  # 5分         │ │
│  │                                 │ │
│  │ def get_weather(city):          │ │
│  │   # キャッシュ確認              │ │
│  │   if cache_valid(city):         │ │
│  │     return cache[city]          │ │
│  │   # API呼び出し                 │ │
│  │   data = api_call(city)         │ │
│  │   cache[city] = data            │ │
│  │   return data                   │ │
│  └────────────────────────────────┘ │
│                                      │
│  📌 エラーハンドリング              │
│  ┌────────────────────────────────┐ │
│  │ try:                            │ │
│  │   response = requests.get(...)  │ │
│  │ except requests.ConnectionError:│ │
│  │   print("ネットワークエラー")    │ │
│  │ except KeyError:                │ │
│  │   print("都市が見つかりません")  │ │
│  └────────────────────────────────┘ │
└──────────────────────────────────────┘
```

### 🎙️ セリフ

**ナレーター:**
```
「実装のハイライトをお見せします。

まずキャッシング機構。
cache という辞書を用意して、
city をキーに、データをバリューで保存します。

CACHE_TTL で有効期限を 300 秒（5分）に設定。
get_weather 関数内で、まずキャッシュを確認して、
有効なら即座に返す。
ないか期限切れなら API を呼び出すというロジックです。

続いてエラーハンドリング。
try-except で複数のエラーをキャッチしています。
ネットワーク接続エラー、都市名が無効なエラー…
実務では こういった細かな対応が必須です。」
```

【設定】
